# PromisedLand.com Data Center Deployment
**Author:** Rose Mayi Mariñez Grullon  
**Course:** IT235 – Cloud Server Administration  

## 📌 Overview  
This project outlines the deployment of a new secondary data center for PromisedLand.com in Independence, Missouri.
It covers server roles, cloud integration, security configuration, storage, DNS, Active Directory, and hybrid management 
to support nationwide call‑center operations.

### 1. Core Server Roles  
- AD DS, DNS, DHCP  
- File & Storage Services (DFS, NTFS)  
- IIS, Hyper‑V  
- Network Policy Server  
- Group Policy Management  

### 2. Cloud & Virtualization  
- Azure AD sync, Azure Backup  
- Hybrid DNS  
- Hyper‑V consolidation + failover  

### 3. Networking  
- Structured IPv4 plan (VLANs/subnets)  
- DHCP scopes + reservations  
- IPAM for address tracking  

### 4. File, Storage & Security  
- DFS for redundancy  
- NTFS permissions & auditing  
- RAID storage + regular backups  
- BitLocker encryption  

### 5. Active Directory & GPO  
- OU structure aligned to departments  
- Password & lockout policies  
- Audit policies for security monitoring  

### 6. Hybrid Management  
- System Center for on-prem servers  
- Azure Portal for cloud resources  
- Load balancing + hybrid backup strategy  

### 7. Monitoring  
- Performance Monitor (CPU, Memory, Disk, Network)  
- Baseline metrics for capacity planning
